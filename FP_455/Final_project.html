Great Smoky Mountains National Park: How terrain and infrastructure influence trail accessibilty GEO 455: Webmapping
Map Layers and Controls

Trail types by difficulty level: Trails separated by difficulty as well as dedicated horse trails. Click on each trail to get the trail name and length.

Search and Navigation: Use trail-name search, the mini map, scale bar, and full-extent button to explore efficiently.

Graduated symbols showing parking areas: Parking symbol size represents the number of parking spots available for each parking lot (still a work in progress).

Physical geography layer: Still a work in progress. I plan to use a DEM of the park and modify the symbology on ArcGIS Pro to help people understand elevation before importing to Phoenix Code as a geoJSON.
